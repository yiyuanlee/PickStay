# PickStay — 90 秒 Demo 脚本

用于面试屏幕共享。超时就砍第 4 步。

| 秒 | 操作 | 口述 |
|----|------|------|
| 0–10 | 打开 https://pickstay.vercel.app | 「PickStay 帮旅行者选**住哪个街区**，不是订酒店。」 |
| 10–25 | 进入东京探索，点「家庭出行」 | 「7 维权重 + 同城归一化排序；卡片上显示驱动维度与分数来源。」 |
| 25–40 | 切换「夜猫一族」 | 「看 Top-1 变化——同一城市，偏好不同，结果不同；这是单测和 E2E 锁住的行为。」 |
| 40–55 | 「复制偏好链接」→ 新标签打开 | 「深链可分享，无需登录就能复现排序，利于传播和 Demo。」 |
| 55–75 | 提一句架构 | 「地图 Key 不上前端：`/api/maps/enrich` + Redis；无 Key 时 Mock 降级。」 |
| 75–90 | （可选）登录 Dashboard / Admin | 「登录后偏好云端同步；Admin 可预热/清缓存，错误会回传到 UI。」 |

**结尾一句：** 「我刻意不做假 AI 推荐——把可观测、安全和可解释工程做到位，数据够了再谈学习排序。」
